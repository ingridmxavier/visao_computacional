🚗 Detecção automática de vagas de estacionamento usando Visão Computacional

Desenvolvi um sistema que identifica vagas livres e ocupadas em um estacionamento a partir de vídeo utilizando OpenCV e Python.

O algoritmo utiliza:
• Conversão para escala de cinza
• Gaussian Blur para redução de ruído
• Threshold adaptativo
• Operações morfológicas
• Análise de pixels por região (ROI)

O sistema processa o vídeo frame a frame e gera um novo vídeo com as vagas detectadas.

Tecnologias: Python, OpenCV, Computer Vision
