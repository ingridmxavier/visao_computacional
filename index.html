🚗 Detecção de Vagas com Visão Computacional

Sistema de identificação automática de vagas livres e ocupadas utilizando OpenCV

Funcionamento do Algoritmo

O sistema utiliza técnicas de processamento digital de imagens para identificar se as vagas do estacionamento estão ocupadas ou livres.

Etapas do processamento

  • Conversão para escala de cinza: A imagem RGB é convertida para grayscale para simplificar o processamento.
  • Gaussian Blur: Aplicação de filtro de suavização para redução de ruído.
  • Threshold adaptativo: A imagem é binarizada para destacar objetos como veículos.
  • Dilatação: Operação morfológica utilizada para reforçar regiões detectadas.
  • Análise por região (ROI): Cada vaga é analisada contando a quantidade de pixels brancos.

Regra de decisão

if pixels_brancos < limite:
    vaga = "Livre"
else:
    vaga = "Ocupada"
